# Benzard Sports Management

This repository is now configured as a `pnpm` workspace with two packages:

- `client`: the Next.js frontend and API routes
- `functions`: Firebase Cloud Functions

## Quick Start

```bash
pnpm install
pnpm dev
```

## Useful Commands

```bash
pnpm build
pnpm lint
pnpm admin:create
pnpm update:dashboard
pnpm functions:deploy
```
